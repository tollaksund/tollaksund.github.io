Flight Demonstration

This image shows the drone during an autonomous indoor navigation test.

Summary: This project involves designing and testing an autonomous drone for a simulated search and rescue mission. The drone must autonomously navigate an indoor environment, detect an action figure (representing a missing person), locate their backpack, and deliver simulated payloads. Using onboard sensors and ArUco marker detection, the drone must demonstrate stable flight, accurate target localization, and efficient path planning, culminating in an autonomous landing. The system is developed over two semesters and tested in QUT’s motion-capture-equipped indoor arena.

Flight test

Video of the live demonstration.

This flight test showcases the drone's ability to autonomously take off, navigate through multiple waypoints, and land within a designated area using onboard sensors and software. It verifies core system capabilities including stable flight, accurate navigation, and integration between hardware and software components.

Aruco Detection

Demonstrates the real-time detection of Aruco marker detection on drone.

ArUco markers are black-and-white square patterns that encode unique IDs, enabling the drone to recognize specific locations in the environment. The onboard camera captures these markers, and computer vision algorithms estimate the position and orientation of each marker relative to the drone. This information is used for tasks like autonomous landing, navigation, and mission execution. Marker detection is performed in real-time using the OAK-D camera and processed through ROS for immediate feedback and control.
