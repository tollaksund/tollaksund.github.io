The PID Challenge: Precision Line Following

This autonomous robotics project focused on achieving **high-speed precision** in path correction using a custom-built chassis and a carefully calibrated **Proportional-Integral-Derivative (PID)** controller. The goal was to minimize lateral error and achieve the fastest possible lap times.

C++ / Microcontroller Primary Algorithm: PID Control Sensors: Infrared Array (8x)

Hardware and Control System

Simulated image of the robot chassis and sensor array.

Core Functionality

The robot uses an array of infrared reflectance sensors positioned beneath the chassis. These sensors provide continuous data on the line's position relative to the center of the robot, translating light intensity differences into a single, quantifiable error value.

This error value is fed directly into the **PID controller**, which calculates the necessary correctional output. This output adjusts the differential speed of the two DC motors, allowing the robot to make rapid, smooth steering adjustments to remain centered over the line, even on sharp curves.

Tuning and Validation:

  • **P (Proportional) Gain:** Provided the initial, fast response to the error.
  • **I (Integral) Gain:** Corrected for steady-state errors (e.g., motor asymmetry) over time.
  • **D (Derivative) Gain:** Dampened oscillations, crucial for smooth high-speed cornering.

Technical Systems Summary

PID Control Logic

A real-time closed-loop control algorithm for dynamic motor differential adjustment, ensuring minimal steady-state error.

Custom Mechanicals

A custom **3D-printed chassis** optimized for low center of gravity and perfect sensor-to-ground distance calibration.

Embedded C++

The firmware was developed in C++ for maximum execution speed and direct hardware manipulation (register access).
