Project Overview

This robot was engineered to precisely follow black electrical tape laid out on a white surface. It uses an array of infrared reflectance sensors to continuously monitor the line position, allowing for rapid and accurate path adjustments.

The core challenge was achieving consistent lap times and managing sharp turns. This was solved through the implementation of a **highly-tuned feedback control system**, demonstrating a practical application of mechatronics principles.

Performance Highlights:

  • Achieved **real-time error correction** based on light intensity data.
  • Validated for consistent lap times under competition rules.
  • Robust performance across varying lighting conditions.

Technical Stack & Hardware

PID Control Logic

Implemented a **Proportional-Integral-Derivative (PID)** controller to dynamically adjust motor speeds. This allowed for smooth, aggressive cornering and minimized overshoot.

Custom Mechanics

Features a **custom 3D-printed chassis** for optimal sensor placement and weight distribution, paired with powerful DC motors and encoders for precise distance measurement.

Microcontroller Firmware

Firmware was developed entirely in **C++** on a high-speed microcontroller to ensure low-latency sensor reading and control loop execution.
