Autonomous Warehouse Robot

A mobile robot designed to navigate and operate in a dynamic warehouse environment.

Overview

This robot was developed to autonomously navigate a warehouse, detect obstacles, and deliver items. Using a combination of ultrasonic sensors, wheel encoders, and line-following techniques, it can operate efficiently in structured environments.

Design Process

The chassis was designed in Fusion 360 and 3D-printed. The electronics were prototyped using an Arduino-based system, and the robot was tested in multiple indoor setups.

Key Features

  • Obstacle detection and avoidance
  • Line-following navigation
  • Delivery compartment with servo control
  • Modular and maintainable chassis design

System Architecture

The robot operates using a structured navigation system divided into three stages: locating the item, grabbing it, and delivering it to the packing bay. Each stage includes substages and is supported by a finite state machine that ensures smooth transitions and recovery if a task fails. The robot uses a model-based reflex agent to make decisions based on both current and past sensor input.

Navigation and Obstacle Avoidance

The navigation system converts sensor inputs into motor outputs, enabling the robot to move, rotate, and avoid obstacles in real-time. It integrates with the vision system to activate only necessary sensors, optimizing performance on the Raspberry Pi. A dual strategy combining potential fields and Bug 2 algorithms allows the robot to maintain a smooth trajectory toward the target while dynamically avoiding obstacles.

  • Potential Fields: Smooth, intuitive navigation toward targets with obstacles treated as repelling forces.
  • Bug 2 Algorithm: Boundary-following strategy to navigate around obstacles when encountered at close range.

Operational Stages

Stage 1 – Item Location: The robot scans for row markers using the packing bay as a reference. It avoids blind zones and stabilizes between shelves for accurate vision input.

Stage 2 – Item Pickup: The robot raises the grabber, aligns with the item, and picks it up using focused visual input.

Stage 3 – Delivery: The robot drives to the packing bay while avoiding obstacles and releases the item, then resets for the next cycle.

Performance and Validation

During tests, the robot demonstrated successful autonomous behavior, consistently retrieving and delivering items. The integration of mobility and vision systems allowed for robust obstacle avoidance and precise navigation. In real-world trials, the robot achieved a success rate of approximately 75%, with minor inconsistencies attributed to lighting, traction, and battery variation.

Reflection

This project helped develop skills in embedded programming, sensor integration, and mechanical design. It also emphasized the importance of robust testing and iterative development.
